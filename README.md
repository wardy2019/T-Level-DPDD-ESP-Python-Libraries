# T Level ESP Task 4a – Pandas & Matplotlib Knowledge Library

This repository contains a **complete knowledge library** for Task 4a of the T Level Digital Production, Design and Development Employer Set Project (ESP). It provides **clear, PEP 8–compliant Python examples with inline comments**, designed to support both **student revision** and **Copilot/AI knowledge agents**.

---

## What’s Included

- **index.html** – A self-contained reference page covering:
  - Importing and reading CSV files with pandas  
  - Selecting, filtering, and creating subsets  
  - Counting, sorting, and finding extremes  
  - Creating new columns  
  - Data cleaning (handling missing values, dropping, renaming)  
  - Grouping, aggregating, and sorting data  
  - Working with dates and identifying trends over time  
  - Error handling using `try/except` (FileNotFoundError, KeyError, ValueError)  
  - Graphing with matplotlib (bar, line, pie) including formatting (titles, labels, legends, readability)  
  - loc and iloc usage  
  - Statistical methods (mean, min, max, sum)  
  - Core syntax reference for quick recall  
  - **Common exam tasks (mapped solutions)** to support decision-making in the assessment  

---

## Purpose

This resource is designed with two key goals:

### 1. Student Support  
A clear, revision-friendly knowledge library that:
- Explains each pandas and matplotlib command with **simple inline comments**  
- Supports both **writing code from scratch** (Task 4a) and **fixing/debugging code** (Task 2)  
- Acts as a **one-stop reference** for independent learning and practice  

### 2. AI / Copilot Integration  
A structured and consistent reference that:
- Can be linked to Copilot agents for **reliable code generation tasks**  
- Ensures student-facing AI tools stay **within specification and expected skill level**  
- Provides consistent syntax examples to reduce errors in generated responses

## How to Use

### Students
- Open `index.html` in a web browser  
- Read each section and use the comments to understand what the code is doing  
- Copy and test examples in VS Code using your own CSV files  
- Use the **Common Exam Tasks section** to help decide what techniques to apply in exam scenarios  

### Teachers
- Share the HTML file or repository link with students  
- Use as a **lesson reference**, **revision guide**, or **intervention support tool**  
- Pair with practical tasks, datasets, or Copilot activities to reinforce learning  
- Extend with additional `.py` files or datasets if needed
